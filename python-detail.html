Brief description on Python

The programming language Python was conceived in the late 1980s, and its implementation was started in December 1989 by Guido van Rossum at CWI in the Netherlands as a successor to ABC capable of exception handling and interfacing with the Amoeba operating system.

Although 30 years old, this programming language is still ruling the computer science landscape and is the hottest programming language used in data science, machine learning, deep learning, back-end development. It is widely used by both computer science and non-computer science people.

Mastering this programming language will unlock a huge array of possibilities in computer science research as well as in multi-disciplinary domains.
